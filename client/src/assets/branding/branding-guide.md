## 3. VISUAL IDENTITY

### 3.1 Color Palette

| Color | Name | Hex | Usage |
|-------|------|-----|-------|
| ⬛ | Midnight | `#0D1117` | Primary background, text |
| 🔵 | Electric Blue | `#0A84FF` | Primary accent, links, highlights |
| 🟢 | Signal Teal | `#00D4AA` | Secondary accent, gradients |
| ⬜ | Clean White | `#F8F9FA` | Body text on dark, backgrounds |
| 🩶 | Slate | `#8B949E` | Secondary text, captions |
| 🟡 | Amber | `#F0A500` | CTAs, urgent highlights, alerts |

**Gradient:**
- Primary gradient: Electric Blue (`#0A84FF`) → Signal Teal (`#00D4AA`)
- Used on: accent elements only (logo mark, highlight bars, buttons)
- Never used on: full backgrounds or large fills

### 3.2 Logo System

| Variant | Description | Use Case |
|---------|-------------|----------|
| **Full Logo** | Diamond shape with integrated play symbol + "PRESSURE PLAY" wordmark | Website headers, intro screens, press kits |
| **Icon Mark** | Compact diamond with play triangle | App icons, social avatars, favicon |
| **Monogram** | PP | Merch, watermarks, thumbnail corners |

**Logo Rules:**
- The diamond mark always uses the blue-to-teal gradient
- Minimum clear space around logo: 1x the height of the diamond mark
- Never stretch, rotate, or recolor the logo outside approved palette
- On dark backgrounds: full-color logo preferred
- On light backgrounds: Midnight-colored wordmark with gradient mark
- Never place the logo on busy or low-contrast backgrounds

### 3.3 Typography

| Typeface | Weight | Usage |
|----------|--------|-------|
| **Inter** | 500 (Medium) | Headings, titles, section headers — tracked spacing |
| **Inter** | 400 (Regular) | Body text — generous line height (1.6–1.8) |
| **JetBrains Mono** | 400 | Episode numbers, timestamps, stats, data overlays |

**Type Rules:**
- Headings: Inter Medium, uppercase or title case, letter-spacing +2%
- Body: Inter Regular, sentence case, generous whitespace
- Data: JetBrains Mono — used anywhere a number, stat, or timestamp appears
- Never use more than these two typefaces in any single piece of content

### 3.4 Visual Language

**Do:**
- Clean flat compositions
- Generous whitespace
- Split-screen imagery (sport on one side, business on the other)
- Data overlays on imagery (stats, metrics, quotes)
- Subtle grid patterns in backgrounds
- Diamond motifs as graphic elements
- Blue-to-teal gradient on accent elements only
- Dark mode aesthetic (Midnight background as default)

**Don't:**
- Busy, cluttered layouts
- Stock photo aesthetic
- Neon or overly saturated colors
- Rounded, playful, or cartoon-style graphics
- Filters that reduce image quality
- Generic podcast cover art styles

### 3.5 Imagery Style

- High-contrast, cinematic photography
- Athletes in motion or intense focus — never posed/cheesy
- Business environments: boardrooms, skylines, screens with data
- Dubai cityscape as recurring visual backdrop
- Always feel premium, editorial, and intentional
- When using both hosts: equal visual weight — neither host is secondary

## 4. BRAND VOICE

### 4.1 Voice Principles

| Principle | What It Means |
|-----------|---------------|
| **Intelligent but accessible** | Never talk down, never gatekeep, but don't dumb things down either |
| **Confident, not arrogant** | Authority comes from insight, not volume |
| **Data-driven but human** | Stats and frameworks matter, but so do stories and emotion |
| **Direct and honest** | Locker room honesty meets boardroom precision |
| **Premium without being pretentious** | Aspirational but grounded |

### 4.2 Writing Style Rules

- Short, punchy sentences. No fluff.
- Active voice always. Cut filler words ruthlessly.
- Numbers and specifics over vague claims:
  - ✅ "67% of athletes fail in their first business"
  - ❌ "Many athletes struggle after retiring"
- Reference both sport AND business in every piece of content — this is our DNA
- Headlines should create tension or curiosity:
  - ✅ "Why Your Business Strategy Needs a Half-Time Talk"
  - ❌ "Business Lessons from Sport"

### 4.3 Words We Use

| ✅ Use | ❌ Avoid |
|--------|---------|
| Edge | Hack (unless in "biohacking") |
| System | Trick |
| Operate | Hustle |
| Execute | Grind |
| Titan | Guru |
| Pressure | Struggle |
| Precision | Perfection |
| Perform | Try hard |
| Arena | Space |
| Deploy | Launch (unless literally launching) |

### 4.4 Brand Phrases

Use these naturally across content — never force them:

- "Pressure creates diamonds" 💎
- "Performance has no off-season"
- "The pitch and the boardroom run on the same playbook"
- "Where the boardroom meets the field"
- "Same pressure, different arena"
- "Your edge is your system"

- "The transfer window is always open"
- "Half-time is for adjustments, not excuses"
- "Every titan was once under pressure"

### 4.5 Emoji Usage

| Emoji | Meaning / Use |
|-------|---------------|

## 6. PLATFORM-SPECIFIC GUIDELINES

### YouTube (Flagship)
- Full video episodes, cinematic quality
- Thumbnails: high-contrast, 2–4 words max, faces showing emotion, diamond motif in corner
- Titles: curiosity-driven, include guest name if applicable, under 60 characters
- Descriptions: SEO-optimized, include timestamps, links, guest info

### Instagram / TikTok (Short-Form)
- 45–90 second vertical clips
- Hook in first 2 seconds — no slow intros
- On-screen text overlays (Inter Medium, white on dark)
- Each clip must have a standalone narrative arc
- 3–5 clips per episode

### X / Twitter
- Thread breakdowns: 8–10 tweets max
- Each tweet stands alone — if someone reads only one, it still delivers value
- First tweet = the hook. Last tweet = CTA
- Quote graphics: Midnight background, Electric Blue accent, Inter typography
- Use polls for engagement

### Spotify / Apple Podcasts
- Full audio, same-day release as YouTube
- Episode art: consistent template with guest photo and episode number in JetBrains Mono

### Newsletter (Weekly — Thursday)
- Editorial quality, clean, concise
- Lead with the insight, not the episode plug
- Sections: Top Insight, Episode Recap, Tool of the Week, One Line from Patrice
- CTA to YouTube episode at the end

## 7. THUMBNAIL GUIDELINES

Every YouTube thumbnail must include:

1. **A face** — host or guest, showing clear emotion (intensity, surprise, confidence)
2. **Text** — 2–4 words maximum. Large, bold, Inter Medium. High contrast.
3. **Brand mark** — PP monogram or diamond icon in bottom corner (small, subtle)
4. **Color** — Midnight background or cinematic image with blue/teal accent element
5. **Split visual** — when topic allows, use the sport-vs-business split-screen concept

**Never:**
- Red arrows, circles, or clickbait graphics
- More than 4 words of text
- Cluttered compositions
- Low-resolution images
